BloGGing CenteR
Платформа для ведения тематических блогов.

Проверка результатов: https://docs.google.com/spreadsheets/d/1n_S58MZZfVKRg1aMpdrzWe29fnXVAYm5FGwkpO8Lnsk/edit?gid=0#gid=0

Установка и запуск
1. Требования
- Установленный XAMPP
- PHP 8.0 
- MySQL 8.0 

2. Установка
Клонируйте репозиторий
git clone https://github.com/r0de0-student/BloGGer-CenteR.git
Переместите содержимое папок в корень веб-сервера
Для XAMPP: C:\xampp\htdocs\blogger-center\
