## BloGGing CenteR
Платформа для ведения тематических блогов.

## Функционал
- Регистрация и авторизация пользователей
- Создание и управление блогами
- Публикация, редактирование и удаление статей
- Система комментариев
- Подписка на блоги
- Админ-панель для управления пользователями

## Технологии
- PHP 8.0+
- MySQL
- PDO (защита от SQL-инъекций)
- HTML/CSS

## Установка
1. Скопируйте папку `blogger-center` в `htdocs`
2. Импортируйте базу данных из файла `database.sql`
3. Настройте подключение в `config/database.php`
4. Откройте `http://localhost/blogger-center/`
